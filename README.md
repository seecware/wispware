## What is Wispware?

Wispware is a **modern, production-grade Docker environment** built for:

- Laravel 12
- Filament
- Vite
- ARM64 systems (Raspberry Pi)

Designed for **serious developers** who want **clean architecture, high performance and zero permission pain**.
